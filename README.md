# Class Percentile & Letter-Grade Estimator

A Python-based command-line tool that estimates a student's percentile rank and expected letter grade based on class distribution data.

## Features

- Customizable grade tiers (A+, A, A-, etc.)
- Editable score ranges for each tier
- Validates class size and tier counts
- Estimates:
  - Letter grade
  - Approximate rank
  - Top percentage
  - Percentile position

## How It Works

The estimator calculates:

1. The number of students in higher tiers
2. Your relative position within your grade tier
3. An estimated rank and percentile based on class size
